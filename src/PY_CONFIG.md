## Environemt configuration
+ #### For a faster development workflow follow the below steps:
    1. `pip install uv`
    2. `cd src`
    3. `uv venv`
    4. `.venv/bin/activate`
    5. `uv sync`

+ #### In case of any issues, follow the standard workflow steps: 
    1. `cd src`
    2. `python -m venv .venv`
    3. `.venv/bin/activate`
    4. `pip install -r requirements.txt`
